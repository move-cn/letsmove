# Task 5 - 实现一个最简单的swap

主要功能是不依赖于第三方的物品交换，且在交换完成前，不允许对物品就行修改（知识点：dof, lock, escrow）

## 创建并项目
### 新建项目
```bash
sui move new myswap
```

### 构建、发布
```bash
sui move build
sui client publish --gas-budget=50000000 
```
得到结果：

packageid: 0x7f65c11f34a281ac3f64f99a24868e2ac76e7936befd7d0c8c47140f90c405a3
