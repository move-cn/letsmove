## Mint my first MFT

```
sui client call  \
--package  0xc3a53ecab42cf15733267b9d9dfa23aaa5b44046ef32d5e57034011b67d71906 \
--module nft \
--function mint_to_sender \
--args 'goldpumpkin nft' 'first nft of goldpumpkin' 'https://avatars.githubusercontent.com/u/22197388?v=4'
```
