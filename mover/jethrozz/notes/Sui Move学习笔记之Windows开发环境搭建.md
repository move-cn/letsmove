点击进入sui的发行页面，下载对应版本的安装包

# 2、配置环境变量

解压第一步下载的压缩包。
打开计算机-> 属性->高级系统设置->环境变量

编辑 '系统变量'  Path ,点击浏览 选择刚刚解压的目录. 确定之后保存即可

# 3、验证

保存完环境变量后,打开命令行,输入sui -V, 出现版本号表示安装成功
